Bio

I'm currently a first-year graduate student in the Database Systems Group at the University of Wisconsin-Madison. I'm grateful to be advised by Dr. Jignesh Patel. Broadly, I'm interested in developing data management systems that run efficiently on current and future hardware architectures, with applications to healthcare and biomedical research.

Software

Hustle
Hustle is an experimental scalable data platform using a relational kernel and a microservices-based architecture. Within the Hustle project, I'm interested in efficient storage management and modular transaction processing.
GOALed (iOS, Android)
Goal attainment scaling is a method that objectively measures effectiveness of rehabilitative therapy. GOALed is a mobile application that provides a user-friendly interface to calculate and interpret goal attainment results.
